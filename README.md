# Latin Materials — Papakosta Tutoring

Interactive learning resources for Latin, organized by function and aligned to common assessment frameworks including the National Latin Exam (NLE).

All materials are built by **Papakosta Tutoring** for use with private students and as part of a broader classical languages curriculum.

## Folders

| Folder | Description |
|--------|-------------|
| [`nle/`](./nle) | National Latin Exam preparation, organized by exam level |
| [`escape-rooms/`](./escape-rooms) | Interactive escape room games for vocabulary and grammar |
| [`quizzes/`](./quizzes) | Interactive quizzes — grammar drills, participles, syntax |
| [`readings/`](./readings) | Materials built around specific Latin authors and texts |
| [`vocabulary/`](./vocabulary) | Word lists, Quizlet TSV files, frequency-based reference |

## How to use

Each folder contains standalone interactive HTML files. To use one:

- **Online:** open the GitHub Pages link for the file
- **Offline:** download the HTML file and double-click it. Runs in any modern browser.

All resources work on desktop, tablet, and phone.
